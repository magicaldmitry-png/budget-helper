# Budget Helper

Консольная программа для учёта и анализа расходов пользователя за месяц

Проект выполнен по дисциплине **«Основы программирования»**

Цель проекта: показать использование простейших алгоритмов и структур данных на примере программы для работы с личным бюджетом

## Описание проекта

Программа **Budget Helper** позволяет пользователю вести учёт расходов за месяц

Каждый расход содержит три параметра:

* день месяца,
* сумма расхода,
* категория расхода

Пользователь может вручную добавлять расходы, загружать тестовые данные из CSV-файла, смотреть список всех расходов, считать сумму расходов за период, находить день с максимальными тратами, анализировать расходы по категориям, отменять последнее добавление и выводить расходы из дерева.

## Основные возможности

Программа поддерживает следующие действия:

1. Загрузка тестовых данных за последние 31 день из CSV-файла.
2. Добавление нового расхода вручную.
3. Вывод всех расходов.
4. Подсчёт суммы расходов за выбранный период.
5. Поиск дня с максимальными расходами.
6. Вывод расходов по категориям.
7. Отмена последнего добавленного расхода.
8. Вывод расходов из дерева в порядке возрастания дней.

## Использованные алгоритмы и структуры данных

* **список** для хранения всех расходов;
* **стек** для отмены последнего добавления;
* **бинарное дерево поиска** для хранения расходов по дням;
* **префиксные суммы** для быстрого подсчёта расходов за период;
* **линейный поиск** для нахождения дня с максимальными расходами;
* **сортировка вставками** для сортировки категорий по сумме расходов.

## Описание модулей

### main.py

Файл является точкой входа в программу.

В нём реализованы:

* вывод меню;
* обработка выбора пользователя;
* ввод данных с клавиатуры;
* загрузка тестовых данных из CSV-файла;
* вызов методов класса `BudgetManager`.

### manager.py

Модуль содержит класс `BudgetManager`.

Этот класс связывает между собой все части программы:

* список расходов;
* стек отмены;
* дерево расходов;
* массив дневных сумм;
* массив префиксных сумм.

### models.py

Модуль содержит класс `Expense`.

Класс описывает один расход пользователя и хранит:

* день месяца;
* сумму расхода;
* категорию расхода.

### algorithms.py

Модуль содержит основные алгоритмы проекта:

* построение массива дневных расходов;
* построение массива префиксных сумм;
* подсчёт суммы расходов за период;
* линейный поиск дня с максимальными расходами;
* подсчёт расходов по категориям;
* сортировка категорий вставками.

### structures.py

Модуль содержит структуры данных:

* `Stack` - стек для отмены последнего добавленного расхода;
* `TreeNode` - узел бинарного дерева поиска;
* `ExpenseTree` - бинарное дерево поиска для хранения расходов по дням.

### expenses.csv

CSV-файл содержит тестовые данные для быстрой демонстрации работы программы.
CSV-файл был добавлен для того, чтобы быстро проверить работу программы на широких данных (79 записей).
Данные для файла были сгенерированы искусственным интеллектом.

Файл имеет структуру:

```csv
day,amount,category
1,28000,жилье
1,950,продукты
2,250,транспорт
```

## Запуск проекта

Для запуска программы нужно открыть папку проекта в терминале и выполнить команду:

```bash
python main.py
```

Если команда `python` не работает, можно попробовать:

```bash
python3 main.py
```

## Особенности реализации

Для подсчёта расходов за период используются **префиксные суммы**

Это позволяет получать сумму за любой период за `O(1)` после предварительного построения массива

Для отмены последнего добавления используется **стек**

Последний добавленный расход удаляется первым

Для хранения расходов по дням используется **бинарное дерево поиска**

При симметричном обходе дерева расходы выводятся в порядке возрастания дней

Для сортировки категорий по сумме расходов используется **сортировка вставками**
